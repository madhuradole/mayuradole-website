# Mayura Dole Website

Static resume website for Mayura M. Dole, built from `CV Mayura Dole 2026.docx`.

## Deploy to Vercel

1. Push this folder to a GitHub repository.
2. In Vercel, import the repository.
3. Keep the framework preset as `Other`.
4. Leave the build command empty.
5. Leave the output directory empty.
6. Deploy.

The original CV is included at `/assets/mayura-dole-cv-2026.docx`.
